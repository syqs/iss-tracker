# ISS Tracker - Angular

## Getting Started

```
$ git clone https://github.com/dYale/node-catapult-angular my-project
$ cd my-project
$ npm install
$ bower install
$ npm start
```

Now visit [localhost:8000](http://localhost:8000/)

### Running the Tests

There is a basic test framework in your `test/` folder. To run the tests, simply run `npm test`.
